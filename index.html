What I Use

These are the tools I use to build and modify the projects I build from small coding challenges to full stack projects. With these resources at my disposal I can make my applications robust, responsive, and modifiable. Check out how I use my skills in my projects section.

C# is the programming language I use to construct the backend of my projects. Using the principles of object-oriented programming and MVC, I can keep my code clean and create efficient, easy to maintain applications.

.NET Core is a Microsoft supported framework for building applications. It provides a plethora of features I use to make my code dynamic, including razor pages, dependency injection, and the ability to host my bigger projects on docker.

I use JavaScript to enhance the functionality on the front end of my applications. It is a very useful tool I can even implement intertwined with C# to find creative solutions to problems. My challenge section is full of javascript applications, so you can see how I utilize it.

I use these coding languages to design the front end of my projects. Bootstrap makes it very easy to add in some components and I can further customize them using CSS. HTML is the core language used for web development of course and I use this to set up the basic design and functionality for my websites.

This is where I store my code for my various projects and challenges in repositories. GitHub makes it easy to check how I coded something previously, and I like the versatility of being able to create a branch and try something completely different without affecting my main code.

I use this as a database for my applications. It allows me to do basic CRUD operations with data as well as store key information. Such as, user information and role designations. I make use of this database in my three larger projects.
